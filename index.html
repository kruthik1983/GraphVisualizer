    Graph Visualizer Tutorial

    Welcome to Graph Visualizer!

    This tool helps you learn graph algorithms through visualization.

    Follow these steps to get started:

    1. Add nodes to the canvas
    2. Connect nodes with edges
    3. Set a start node
    4. Select an algorithm
    5. Run the visualization

    Adding Nodes

    Click the "Add Node" button or press 'N', then click on the canvas to place nodes.

    You can drag nodes to reposition them.

    Adding Edges

    Click the "Add Edge" button or press 'E', then click on two nodes to connect them.

    You can click on an edge's weight to edit it.

    Searching Nodes

    Use the search box in the top right to find nodes by ID or label.

    Matching nodes will be highlighted on the canvas.
